diazotheme.colonialart Installation
-----------------------------------

To install diazotheme.colonialart using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``diazotheme.colonialart`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        diazotheme.colonialart
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
